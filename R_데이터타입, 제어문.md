## 기본 자료형
   
   ### 자료구조
     - vector(벡터)
         : 단일값들의 모임, 동일한 자료형을 갖는 값들의 집합
     - vector 생성함수
         Ex. c(1, 2, 3), rep( ), seq() 
         
     * 의미 해석

     - (- Inf, Inf) : 음과 양의 무한대

     * 자료형 확인 함수
       : is.tnteger(x)  → TRUE or FALSE

dan <- 2

while (i < 10) {
  times <- i * dan
  print(paste(dan, "X", i, " = ", times))
  i <- i + 1
}
